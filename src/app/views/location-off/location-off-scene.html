You must turn on both GPS and Google Services to use this app, or you can enter an address manually by selecting "Enter Address" in the app menu.
Location Services can be managed using the Location Services preference app.
Enabling GPS and Google Services can be done in the menu of the app. If they are both checked and you still get this message, try turning them both off and then back on to see if that solves the issue. (You will also need to restart this app.)
